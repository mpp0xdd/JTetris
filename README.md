# JTetris
JTetris is Tetris written in Java.

# How to compile
## Step1
```
$ git clone https://github.com/mpp0xdd/JGLib.git
$ git clone https://github.com/mpp0xdd/JTetris.git
$ cd JTetris
```
## Step2
### Linux
```
$ javac -d classes -encoding UTF-8 -cp ../JGLib:. *.java
```
### Windows
```
$ javac -d classes -encoding UTF-8 -cp ../JGLib;. *.java
```

# How to run
```
$ java -cp classes JTetris
```

# To play BGM
Prepare sound files and place them in the classes/sounds folder.
